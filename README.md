# tink-web-project
## Running the app
### Backend
1. cd tink-backend
2. npm install
3. add .env variables (CLIENT_ID & CLIENT_SECRET)
4. node app.js

### Frontend
1. cd tink-frontend
2. npm install
3. npm run serve
